01 / DISCOVER

The sidecar watches the firehose

A single shared discovery process polls GeckoTerminal, DexScreener, the bonding-curve flow — plus an optional real-time Bitquery stream — and publishes one clean snapshot of fresh candidates to dat boi. One poller, no rate-limit pile-ups, no babysitting.

02 / SCORE

Every candidate gets a number

Dat boi scores every candidate across dimensions — liquidity depth, momentum, volume acceleration, holder shape. And the score isn't a vibe: the forward return of every scored token is logged, so the system can later check whether its own scoring predicted anything at all.

03 / ADMIT

Most tokens are rejected

Admission gates filter the tape hard — too shallow, too stale, dead pool, rug pattern, quarantined. The bot is selective by design, and the SCOUT view shows you every skip with its reason. Nothing is hidden behind a win reel.

04 / EXECUTE

Sized from the live wallet

An admitted candidate is sized from the SOL actually in that bot's wallet — small balance, small trades — and executed through Jupiter routing with slippage guards. No leverage, no margin, no pooled funds. Your wallet is the whole story.

05 / EXIT

Every close has a reason

Stops, trailing exits, take-profit bands, rug-and-drain guards, dead-pool exits. The ledger records why each position closed — so you can audit every decision after the fact, including the ones that lost.

06 / LEARN

The honest part

Realized, fee-inclusive results feed a learning loop. Rules that prove out get promoted. Sizing stays locked until an edge clears a pre-registered bar — and a kill-criterion can declare an edge disproven and disarm it again. The system would rather stay small than fool itself.

03 · The truth

Most bots show you wins.
DATBOI shows you skips.

The most valuable thing in this codebase isn't a strategy — it's the discipline around it. Every claim the system makes about itself is checkable from your own ledger.

▸
Forward-return loggingEverything scored gets its outcome recorded — including the tokens it didn't buy.
▸
Fee-inclusive accountingEdges are judged on realized proceeds after fees and slippage — not on quotes.
▸
A pre-registered kill criterionThe bar for "this edge is real" is written down before the data comes in. No goalpost moving.
▸
The one ruleSizing is armed only by the gates. Never by hand. Never by hype.

04 · Start your own boi

Here come dat boi —
clone to dashboard in minutes

Three commands and one free API key. The dashboard walks you through the rest — or just ask your Claude to do it.

1
git clone https://github.com/Onasidequestt/datpipboi && cd datpipboi
Grab the code.
2
python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt
Set up Python (3.9+) and install six dependencies.
3
./run.sh
Opens the dashboard at localhost:8080. Paste your free Helius key on the setup page, hit Activate — dat boi generates a fresh wallet, shows you the address to fund, and starts rollin.

Prefer not to touch a terminal? Open the repo in Claude Code and say "set this up for me." The bundled guide gives it everything it needs.
